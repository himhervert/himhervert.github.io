Historia

1960-1969

Estados Unidos crea una red militar, con el objetivo de tener acceso a información militar desde cualquier punto del país siendo cuatro equipos de diversas universidades conectados.

1973

Para 1973 Inglaterra y Noruega se unen a ARPANET cada una con un equipo.

1981

Se define el protocolo TCP/IP y la palabra Internet.

1990

En el Centro Europeo de Investigaciones Nucleares (CERN) Tim Berners Lee retoma un proyecto llamado "Xanadú" el cual usababa hipertexto.
Desaparece ARPANET.
En este año se le dio un nombre World Wide Web (www).

1991

Es anunciada publicamente la World Wide Web.
