Descripción

Nuestra propuesta de proyecto es desarrollar un sistema donde un administrador pueda subir imágenes e información a una base de datos y ser mostradas en una página web, la cual podrá visualizarse en diversos dispositivos con acceso a internet.
Se tiene como objetivo a corto plazo es ser implementado en los laboratorios UDICEI; a largo plazo ser implementado en diversas facultades.

Requerimientos

  • Crear, editar, eliminar anuncios.
  • El sistema debe de ser capaz de subir n imágenes.
  • Subir imágenes de manera sencilla.
  • Mostrar Fecha, Hora, 2 paneles para mostrar imágenes distintas.
